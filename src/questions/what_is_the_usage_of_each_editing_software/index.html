What is the usage of each editing software?

There are many different editors one can use to contribute to OSM. The default editor is iD, which is opened if you click on edit in openstreetmap.org. There are two different useful metrics to compare the editors. The first one is how many users are using the editor to contribute and the the seconde one is how many edits are made with an editor per month or year. For each metric the rank of each editor will differ. There are some editors like iD or StreetComplete that are used by many people to make many smaller important changes and there are bulk editors like JOSM where users can upload external data and make many edits at once.

Looking at how many percent of all contributors useed an editor in a give month, we can get a better feeling of the distribution of used editors.

It's also interesting to look at contributor count of less popular editors.
