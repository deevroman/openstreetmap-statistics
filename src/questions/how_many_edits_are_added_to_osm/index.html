How many edits are added to OSM?

An edit is a change made to a node, way or relation. This way adding the name to a school counts as one edit and adding a new building with multiple nodes count as mulitple edits. This means that edits that change geometry are likely overrepresented over edits that add or change information on existing geometry. Look at methodology for more information.
